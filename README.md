# ASME UF Spring 2026 Design Project: Joy-Con Controlled Robotic Arm

## Overview
This project is a **three-servo robotic arm** designed and built by the University of Florida ASME team for the Spring 2026 design course. The arm is controlled using a **Nintendo Joy-Con**, allowing precise and intuitive manipulation for a variety of tasks.  
This project will be a beginner friendly course designed to prepare young engineers with basic engineering skills in order to join any of the ASME design teams 

### Features
- Three degrees of freedom using servo motors
- Real-time control with a Joy-Con controller
- Simple, intuitive programming with insightful comments
